import Matchers._

eventually retries the past block until it no longer throws an exception, using a timeout and interval taken from an implicit PatienceConfig. The default PatienceConfig is 150 milliseconds timeout and 15 milliseconds interval. If you import IntegrationPatience, you'll get 15 seconds timeout and 150 milliseconds interval. If you want something else, you can define your own implicit PatienceConfig like the next example. 
import concurrent.Eventually._
import org.scalatest.time.SpanSugar._
implicit val patienceConfig =
PatienceConfig(timeout = scaled(2 seconds), interval = scaled(5 millis))
