ScalaTest/Scalactic 3.2.6 Release Notes

ScalaTest/Scalactic 3.2.6 (for Scala 2.10, 2.11, 2.12, 2.13, 3.0.0-RC1; on the JVM, JavaScript, and native) includes the enhancements and bug fixes listed below.

For information on how to include ScalaTest in your project, see the install page. For information on how to use Scalactic in your production code, see its install page.


ScalaTest

ScalaTest 3.2.6 is a minor release of ScalaTest that makes few changes, and supports Scala.js 1.3.0 and Scala 3.0.0-RC1.

Bug Fixes

Load the new correct wrapper class for JUnit and TestNG. (#1974)

Improvements

Changed to use scala-xml 2.0.0-M5 for Scala 3 build and 1.3.0 for Scala 2 build. (#1974)

Deprecations

ScalaTest 3.2.6 contains no new deprecations.


Scalactic

Scalactic 3.2.6 contains no changes.
