ScalaTest 2.1.3 Release Notes

ScalaTest 2.1.3 is a maintenance release for 2.1.x series that includes 5 enhancements and a bug fix. No source code using ScalaTest 2.0+ should break, but you will likely need to do a clean build to upgrade.

Enhancements

  • Enhanced Prettifier.default to handle scala.xml.NodeSeq and scala.xml.NodeBuffer.
  • Enhanced Prettifier.default to handle object that has class type Fred extends Seq[Fred].
  • Refactored Prettifier.default to use stringPrefix for better performance.
  • Added -t argument support to SBT integration.
  • Disallow any suffix for arguments that does not take any suffix.

Bug fixes

  • Fixed owner chain problem in assert macro.

Please visit ScalaTest Release Notes for previous version release notes.
