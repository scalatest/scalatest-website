ScalaTest 2.1.1 Release Notes

ScalaTest 2.1.1 is a maintenance release for 2.1.x series that includes a feature enhancement and a bug fix. No source code using ScalaTest 2.0+ should break, but you will likely need to do a clean build to upgrade.

Enhancements

  • Added -z argument support to SBT integration.

Bug fixes

  • Added the missing must support to noException, a[SomeException] and an[Exception] matcher syntax.

Please visit ScalaTest Release Notes for previous version release notes.
