ScalaTest/Scalactic 3.0.4 Release Notes

ScalaTest/Scalactic 3.0.4 (for Scala 2.10, 2.11 and 2.12) includes the enhancements and bug fixes listed below. No source code using ScalaTest/ScalaUtils 3.0 or higher should break, but from any version other than other 3.0.x releases (with which this release is binary compatible), you will likely need to do a clean build to upgrade.

For information on how to include ScalaTest in your project, see the install page. For information on how to use Scalactic in your production code, see its install page.

Enhancements

  • Updated scala-js version to 0.6.16.

Bug fixes

  • Avoid race condition in Timed.scala. View details here: github issue #1147

Visit ScalaTest Release Notes for links to the release notes of all previous versions, or step back in time by visiting the release notes for the previous version.
