ScalaTest + JUnit 5

The ScalaTest + JUnit 5 integration library makes it fun and easy to use JUnit 5 with ScalaTest. To use ScalaTest + JUnit 5, please add the following to your SBT project dependency: 

libraryDependencies += "org.scalatestplus" %% "junit-@latestJUnit5Version" % "@latestVersion.0" % "test"

For maven you can use: 

<dependency>
    <groupId>org.scalatestplus</groupId>
    <artifactId>junit-@{latestJUnit5Version}_@{majorMinorScalaVersion}</artifactId>
    <version>@latestVersion.0</version>
    <scope>test</scope>
</dependency>
}
