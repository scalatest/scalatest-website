ScalaTest Plus

The ScalaTest Plus project aims to define integration libraries between ScalaTest and other libraries, to make it easier to test code using those libraries. Each integration library will depend on a version of ScalaTest as well as a version of the target library.

Currently, the following ScalaTest Plus libraries are available:

  • ScalaTest + Play (Versions)
  • ScalaTest + Selenium (Versions, Scaladoc)
  • ScalaTest + ScalaCheck (Versions, Scaladoc)
  • ScalaTest + JUnit 4 (Versions, Scaladoc)
  • ScalaTest + JUnit 5 (Versions, Scaladoc)
  • ScalaTest + TestNG (Versions, Scaladoc)
  • ScalaTest + EasyMock (Versions, Scaladoc)
  • ScalaTest + JMock (Versions, Scaladoc)
  • ScalaTest + Mockito (Versions, Scaladoc)
